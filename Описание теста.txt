1. Есть приложение для Android (конвертер плюс)
2. Нужно его установить на любое доступное окружение (телефон, эмулятор)
3. Это приложение для своей работы запрашивает курсы валют из сети
4. Нужно отловить запрос, который отправляет приложение на сервер Европейского банка для получения курса валют
5. Послать через веб-браузер этот запрос и получить в ответе xml с курсами валют
6. В ответе на дз прислать xml-файл с курсами валют
